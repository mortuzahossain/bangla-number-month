Change All Number and Month Name from anywhere of your wordpress into Bangla language.

== Description ==

This is just any wordpress site English language all numbers , date and time convert to bangla language.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload \`Bangla Number And Month\` to the \`/wp-content/plugins/\` directory
1. Activate Bangla Number And Month through the 'Plugins' menu in WordPress
